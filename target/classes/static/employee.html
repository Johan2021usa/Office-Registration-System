Department Home Vehicle

Employee register system





!Please! Insert an employee's id to search it in the system.


With this application you can register your employees into a database or see a list of registers, as well as delete them or update their information,
and last but not least, you can search any employee if you know his id.

Front-end: JavaScript, HTML, CSS
