!Please! Insert an employee's id to search it in the system.


Functional Description

With this application you can register your employees into a database or seeing a list of registers, as well as removing or updating their information,
and last but not least, you can search any employee if you know his id.

Technical Description

The backend of this application is based on a REST API built with Spring Boot, this API allows you to save and retrieve information from MySQL as a database system, the architecture pattern used for this API was the three tire (three layer) architecture. In the other hand, this API rest is being hosted in Oracle Cloud Virtual Machine, and it's being consumed with fetch that makes part of JavaScript

Technologies used

back-end: Spring Boot (Lombok, Devtools, MySQL driver, Spring Web), Spring Data JPA (Hibernate), MySql, Java, Maven

Front-end: JavaScript, HTML, CSS
