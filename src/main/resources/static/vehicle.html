Vehicle Registration

Overview

This system allows you to implement a registration system for an office by including three main components: Employees, Departments and Vehicles. It allows you to save, update, delete and see registers based on your needs. This software incorporates an amiable interface easy to use

Technical Description

The backend of this application is based on a REST API built with Spring Boot, this API allows you to save and retrieve information from MySQL as a database system, the architecture pattern used for this API was the three tire (three layer) architecture. In the other hand, this API rest is being hosted in Oracle Cloud Virtual Machine, and it's being consumed with a web application based on Vanilla Javascript.

Technologies for back-end:

The next are the technologies used to develop the back-end for this system: Spring Boot 2.7.2 (Lombok, Devtools, MySQL driver, Spring Web), Spring Data JPA (Hibernate), MySql, Java 11, Maven 3.9.6

Technologies for front-end:

The next are the technologies used to develop this web application that consumes the API: Vanilla JavaScript, HTML5, CSS3
